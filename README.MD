# Rust RSS阅读器 for Windows 10

一个轻量级、高性能的RSS阅读器，具有现代化的用户界面和流畅的用户体验。
我没学过RUST,此项目完全由AI完成!

## 版本信息

当前版本: 1.3.1

## 技术栈

- **编程语言**: Rust 2024
- **GUI框架**: egui/eframe
- **HTTP客户端**: reqwest,headless_chrome
- **RSS解析**: rss crate
- **数据存储**: DuckDB
- **异步运行时**: tokio
- **系统托盘**: tray-item
- **通知系统**: notify-rust
- **HTML处理**: html2text, html-escape
- **OPML支持**: opml crate

## 核心功能

### RSS源管理
- 添加、删除、编辑RSS源
- RSS源分组分类管理
- 导入/导出OPML文件
- 支持RSS源自动更新配置
- 支持每个源单独启用/禁用通知

### 文章阅读
- 三栏布局：源列表 → 文章列表 → 内容区域
- 文章标题、发布时间、来源显示
- 支持基本HTML内容渲染
- 标记已读/未读状态
- 收藏重要文章
- 批量操作（删除、标记已读）

### 内容获取
- 自动定时更新（可配置间隔）
- 手动刷新单个或所有源
- 后台静默更新
- 支持并发更新多个源

### 用户界面
- 暗色/亮色主题切换
- 响应式设计
- 字体大小调整
- 系统托盘最小化
- 新文章桌面通知

### 高级功能
- 全文搜索功能
- AI自动翻译文章内容
- 支持多种编码格式
- 数据库导出功能
- 键盘导航支持

## 性能要求

- 启动时间 < 2秒
- 支持100+ RSS源同时更新

## 系统要求

- Windows 10 或更高版本
- 64位操作系统

## 安装与运行

### 直接运行

1. 下载最新的发布版本 `rust_rss_reader.exe`
2. 双击运行即可

### 从源代码构建

1. 确保已安装 Rust 开发环境（建议使用 Rust 1.70+）
2. 克隆仓库：
   ```bash
   git clone https://github.com/chtcrack/rust-rss-reader.git
   cd RSS_Rust_Reader
   ```
3. 构建并运行：
   ```bash
   cargo run --release
   ```

## 使用说明

### 添加RSS源

1. 点击左侧面板的「添加源」按钮
2. 输入RSS源的标题和URL
3. 选择或创建分组
4. 点击「保存」按钮

### 阅读文章

1. 在左侧面板选择一个RSS源或分组
2. 在中间面板选择一篇文章
3. 在右侧面板阅读文章内容
4. 使用工具栏按钮标记已读/未读或收藏

### 搜索文章

1. 在顶部搜索框输入关键词
2. 系统会自动过滤显示匹配的文章
3. 点击「清除」按钮重置搜索

### 导出数据

1. 点击菜单栏的「文件」→「导出数据」
2. 选择导出格式（SQL或OPML）
3. 选择保存位置
4. 点击「导出」按钮

## 配置说明

应用程序的配置文件位于用户目录下的 `.rust_rss_reader` 文件夹中。

主要配置项包括：

- 自动更新间隔
- 主题设置
- 字体大小
- 系统托盘显示
- 控制台显示
